System Architecture

How data flows

From physical detection to database — every step of the pipeline, animated in real time.

01 / DETECT
Camera Trigger
Tapo C200 detects motion. MediaPipe activates face landmark tracking.
02 / SENSE
MQ-3 Capture
3-sensor array logs a 2–3 second vapor window. Spatial gradient computed.
03 / EXTRACT
Feature Extraction
Rise time, peak, decay slope, spatial variance — 14 statistical features total.
04 / CLASSIFY
ML Inference
Random Forest + XGBoost ensemble fuses sensor features with EAR. 3-class output.
05 / LOG
Store & Alert
Result written to MariaDB. Dashboard notified via FastAPI WebSocket. Alert if Class 1.

System Capabilities

Built for real-world use

01
3-Sensor Spatial Array
Three MQ-3 sensors form a spatial grid, enabling vapor direction tracking and gradient analysis across the entrance zone.
Hardware
02
Computer Vision EAR
MediaPipe Face Mesh tracks 468 landmarks in real time. Eye Aspect Ratio detects impairment over 30-frame rolling windows.
CV / ML
03
Fusion Classification
Random Forest + XGBoost ensemble fuses 14 sensor features with EAR for robust 3-class event classification at 97.2% accuracy.
Machine Learning
04
False Positive Prevention
Multi-frame EAR analysis, 5-reading rolling averages, and dual-source validation eliminate false alarms from sanitizer or perfume.
Reliability
05
Fully Embedded
FastAPI + React + MariaDB all run on Raspberry Pi 4. Zero cloud dependency. Full offline operation in any environment.
Embedded
06
Face Recognition
DeepFace + FaceNet identifies registered subjects and tracks detection history per person across sessions.
